## 选型原则

- **Mac 原生优先**：mlx > PyTorch+MPS，减少 fallback 开销
- **零训练成本优先**：Mac 本地训练 > 云 GPU 租用
- **小而精**：0.5B-3B 学生模型，分类/抽取/简单生成任务
- **蒸馏数据壁垒**：教师模型生成的高质量垂直数据是核心壁垒
